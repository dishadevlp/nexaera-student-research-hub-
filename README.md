This repository contains the official website for Nexaera, developed to create a simple and accessible digital presence for the initiative. The website provides information about Nexaera, its purpose, and future activities.
Nexaera is a student-led initiative focused on building platforms, ideas, and opportunities that support learning, collaboration, and practical skill development.
This project was created as part of a hands-on learning journey to understand how websites are built, deployed, and improved over time using modern tools.
